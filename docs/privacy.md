# Privacy

Recognize is designed as a local-first desktop helper.

Default behavior:

- Screenshots are processed locally.
- OCR text is not uploaded.
- Real candidate data should not be committed to GitHub.
- Debug screenshots are intended for local troubleshooting only.

Before sharing logs, screenshots, or GitHub issues, remove:

- Candidate names.
- Phone numbers.
- Email addresses.
- Resume text.
- Chat messages.
- Company-private identifiers.
